# 文瀛餐厅管理系统 — MVP 版本

> 10 天交付版本，专为单人/小团队、2 年编程基础的同学准备。
> 核心理念：**能用够用、闭环优先、答辩讲得清**。

## 文件清单

| 文件 | 用途 |
|------|------|
| [MVP_设计与10天计划.md](MVP_设计与10天计划.md) | 核心文档：简化版需求 + 技术栈 + 数据库 + 接口 + 10 天日程 + 答辩话术 |
| [schema.sql](schema.sql) | 数据库建表 + 演示数据，直接复制执行 |

## 一句话定位

> 一个**管理员账号**、**4 张表**、**10 个接口**、**5 个页面**、**4 张图表** —— 完成"上架菜品 → 模拟下单 → 自动扣库存 → 看板可视化"完整闭环。

## 推荐顺序

1. 先读 [MVP_设计与10天计划.md](MVP_设计与10天计划.md) 的第 0、1、7 节，建立整体认知
2. D1 开始时执行 [schema.sql](schema.sql) 建库
3. 严格按照 10 天日程推进，每天对照"自检表"
4. 答辩前必看第 8 节"答辩防忽悠话术"
