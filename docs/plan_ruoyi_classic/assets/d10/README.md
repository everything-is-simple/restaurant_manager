# D10 截图目录

本目录用于保存答辩截图，推荐文件名如下：

| 文件名 | 内容 |
|--------|------|
| `01-login.png` | 登录页品牌 |
| `02-dashboard.png` | 首页总览 |
| `03-recipe.png` | 菜品配方页 |
| `04-stock-in.png` | 入库管理页 |
| `05-order.png` | 订单管理页 |
| `06-stats.png` | 数据看板页 |
| `07-role-manager.png` | 食堂经理菜单 |
| `08-role-stockkeeper.png` | 库管员菜单 |

当前目录已补齐上述 8 张真实页面截图，文件名与用途一一对应。

2026-06-21 本轮官方应用内 Browser / Computer Use 会话仍返回 `missing field sandboxPolicy`，因此未采用官方自动化链路。本轮截图改由本机可用的 Edge/CDP 真页面方式补齐，不影响答辩材料使用。

使用说明：

- `01-login.png`、`02-dashboard.png` 用于品牌展示。
- `03-recipe.png`、`05-order.png`、`06-stats.png` 用于 manager 主链讲解。
- `04-stock-in.png`、`08-role-stockkeeper.png` 用于库管员职责边界讲解。
- `07-role-manager.png`、`08-role-stockkeeper.png` 一起使用，可直接证明角色菜单隔离。
